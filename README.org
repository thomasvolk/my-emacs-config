* Common Commands:

| C-x C-f | Open file browser      |
| C-x C-c | Quit emacs             |
| C-x o   | Switch between windows |

* Org Mode Commands

| TAB     | Un/Fold a section    |
| <s TAB  | Insert code block |
| C-c C-c | Execute a code block |

#+end_src
